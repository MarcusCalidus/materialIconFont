# install
## install node
https://nodejs.org/en/

## install package dependencies
(Make sure you run in the same directory as svgToFontScraper.js)
- `npm install`

# making fonts
1. Replace svgs located in ices_driver or icns_rider
2. Navigate and make sure directory is in same location as svgToFontScraper.js (in terminal using CD)
3. `node svgToFontScraper.js` in the terminal
4. Bin will hold all generated assets
5. Upload it to its corresponding google drive folder (try not to replace the parent folder cause it'll change the unique link on the platform websites)
